# Northcoders News API

## Set Up


<p>Create two <strong>.env</strong> files locally in order to access the database</p>


<ol>
 <li> .env.development should contain:</br>
  <code>
    PGDATABASE=nc_news
  </code>
  </li>
  <li>.env.test should contain:</br>
  <code>
PGDATABASE=nc_news_test
  </code>
  </li>
  </ol>
